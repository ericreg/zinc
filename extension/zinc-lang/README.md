# Zinc Language Support

Basic VS Code language support for Zinc `.zn` files.

## Features

- Syntax highlighting for comments, attributes, strings, string interpolation, numbers, enums, structs, Zinc imports, Rust extern blocks, metadata builtins, type annotations, operators, and punctuation.
- Bracket matching and auto-closing pairs for braces, brackets, parentheses, and quotes.
- Line and block comment configuration.

## Development

Open this folder in VS Code and press `F5` to launch an Extension Development Host. Open a `.zn` file there to test the grammar.
